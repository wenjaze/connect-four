## Connect Four

- java implementation of the classic [Connect Four](https://hu.wikipedia.org/wiki/Connect_four) game
- used technologies:
  - openjfx-16
  - maven
  - delombok
  - junit
  - jdbi
  - log4j
  - javadoc
  - h2
  - hibernate
  - common-langs
