## Connect Four

- java implementation of the classic [Connect Four](https://hu.wikipedia.org/wiki/Connect_four) game
- used technologies:
  - openjfx-16
  - maven
  - delombok
  - junit
  - jdbc
  - log4j
  - javadoc
  - h2
  - hibernate
  - common-langs
- [*official requirements*](https://arato.inf.unideb.hu/jeszenszky.peter/download/swe/swe-spec_hu.pdf)
